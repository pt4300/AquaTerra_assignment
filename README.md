### Installation


1. Clone the repo
   ```sh
   git clone https://github.com/pt4300/AquaTerra_assignment.git
   ```
2. cd to source code folder
	   ```sh
   cd dog_demo
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. start the npm server
   ```sh
   npm start
   ```
